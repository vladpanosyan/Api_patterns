FOR EXAMPLE

ts-node ./src/index.ts add bulldog --count=2 (count default is 3)
Aded in breed.json file the cloud store adress where was saved "bulldog's" images which count is "2" regardless of subbred of bulldog

Or

ts-node ./src/index.ts add bulldog english
Aded in breed.json file the cloud store adress where was saved "bulldog's" images which count is "3" (Default is 3) and subbred is "english"

After than

Run in command prompt npm run serv

Go to "localhost:3000/bulldog"

Go to "localhost:3000/bulldog/english"
